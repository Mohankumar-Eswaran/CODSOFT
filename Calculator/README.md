# Calculator

## Description
A simple calculator application that performs basic arithmetic operations. This project uses Python and Tkinter to create a GUI-based calculator.

## Features
- Addition, subtraction, multiplication, and division.
- Clear the current input.
- Delete the last digit.

## Usage
1. Run the `calculator.py` script.
2. Use the GUI to perform calculations.

## Requirements
- Python 3.x
- Tkinter

## Run the application
```bash
python calculator.py
