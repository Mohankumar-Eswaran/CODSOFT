# Rock-Paper-Scissors Game

## Description
A simple Rock-Paper-Scissors game. This project uses Python and Tkinter to create a GUI-based application.

## Features
- User can choose rock, paper, or scissors.
- Computer generates a random choice.
- Determine the winner based on the user's and computer's choices.
- Display the result and score.

## Usage
1. Run the `rock_paper_scissors.py` script.
2. Use the GUI to play the game.

## Requirements
- Python 3.x
- Tkinter

## Run the application
```bash
python rock_paper_scissors.py
