# To-Do List Application

## Description
A To-Do List application to help users manage and organize their tasks efficiently. This project uses Python and Tkinter to create a GUI-based application that allows users to create, update, and track their to-do lists.

## Features
- Add new tasks with a title, description, due date, and priority.
- Update existing tasks.
- Delete tasks.
- Mark tasks as completed.
- Save tasks to a file.
- Load tasks from a file.

## Usage
1. Run the `to_do_list.py` script.
2. Use the GUI to manage your tasks.

## Requirements
- Python 3.x
- Tkinter

## Run the application

python to_do_list.py
