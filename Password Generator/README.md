# Password Generator

## Description
A password generator that creates strong and random passwords for users. This project uses Python and Tkinter to create a GUI-based application.

## Features
- Specify the length of the password.
- Generate a password with a combination of random characters.

## Usage
1. Run the `password_generator.py` script.
2. Use the GUI to generate passwords.

## Requirements
- Python 3.x
- Tkinter

## Run the application
```bash
python password_generator.py
